# PyPic Editor Plus

A powerful Python-based image editor with advanced analysis capabilities.

## Features
- **Editing**: Brightness, Contrast, Rotation, Crop, Skew, Filters (Blur, Sharpen, etc.)
- **Analysis**: Edge detection, Noise analysis, Histogram comparison, Metrics (PSNR, SSIM), and Artifact detection.
- **UI**: Modern Tkinter interface with real-time preview.

## Installation
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
Run the application:
```bash
python main.py
```
